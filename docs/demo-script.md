# Demo Script (Presentation)

This script is designed for a 7–12 minute live demo.

## Setup (before presenting)
- Ensure at least: 1 admin user, 2 officers (with designation), a few offenders/cases, one active program.

## 1) Admin login + system overview (1 min)
- Log in as admin.
- Show dashboard summary cards (users/offenders/cases/programs).

## 2) Officer management (1–2 min)
- Navigate to Officer list.
- Create or open an officer profile and show **designation**.
- Open officer detail/caseload page and highlight assigned cases/offenders.

## 3) Offender supervision (2–3 min)
- Open an offender profile.
- Show case information and assigned officer.
- Create/update a case if needed.

## 4) Auto-assign officers (1 min)
- In admin: select unassigned active cases → run “auto-assign” action.
- Explain: least-loaded strategy for fair distribution.

## 5) Monitoring (1–2 min)
- Schedule a check-in for a case.
- Mark it completed or missed and show compliance fields.

## 6) Programs (2–3 min)
- Create or view a program category and a program (show code, frequency, delivery method).
- Enroll an offender in a program.
- Show sessions and take attendance.
- Point out completion rate and participant tracking.

## Wrap-up (30 sec)
- Summarize how the system standardizes supervision, monitoring, and rehabilitation outcomes.
